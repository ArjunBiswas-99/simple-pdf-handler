# Simple PDF Handler
by Arjun Biswas

A Python-based desktop application that provides a clean and intuitive interface for comprehensive PDF manipulation. The application enables users to perform a wide range of tasks including viewing, editing, annotating, converting, and managing PDF documents.

## Features

### Viewing & Navigation
* Open and render PDF files from local storage
* Navigate pages (next, previous, first, last, specific page number)
* Zoom in/out and adjust layout (fit width, fit page)
* Search for text within the document
* Select and copy text/images

### Editing & Modification
* Add and edit text (font, size, color, style)
* Insert, resize, move, and delete images
* Manipulate objects (move, resize, change properties)

### Page Management
* Insert pages from other PDFs or blank pages
* Delete, rotate, reorder, crop, and extract pages
* Replace pages with content from other documents

### Annotation & Markup
* Add highlights, underlines, and strikethroughs
* Create sticky notes and comments
* Add text boxes and callouts
* Draw shapes and lines
* Add standard stamps

### Merging & Combining
* Merge multiple PDF files into a single document
* Insert specific pages from one PDF into another

### Conversion
* Convert PDFs to Word, Excel, PowerPoint, and plain text
* Convert PDFs to image formats (JPEG, PNG, TIFF)
* Convert PDFs to HTML
* Create PDFs from Word, Excel, PowerPoint, images, and web pages

### Advanced Features
* Optical Character Recognition (OCR) for scanned documents
* Basic accessibility checking for PDF documents
